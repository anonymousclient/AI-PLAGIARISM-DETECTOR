AI Assignment Plagiarism Checker

A web-based system designed to help educational institutions maintain academic integrity. Faculty can create assignments, students upload their submissions (PDF/DOCX), and the system automatically detects plagiarism using TF-IDF similarity analysis and optional AI-based analysis via Gemini API.

Purpose
  • Automate plagiarism detection for academic assignments
  • Provide faculty with visual, color-coded similarity reports
  • Reduce manual effort in checking copied submissions
  • Promote original work and academic honesty
Technologies Used
Flask MongoDB Atlas Scikit-learn (TF-IDF) Bootstrap 5 PyPDF2 python-docx Gemini API
